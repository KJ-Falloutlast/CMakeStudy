# How to push local data to github
1. 第1次上传步骤：
>1. git init 
>2. git add .
>3. git commit -m "注释语句"
>4. git push -u origin main
>5. git remote set-url origin  https://<your_token>@github.com/<USERNAME>/<REPO>.git
>6. git push

2. 第N次上传步骤

>1. git init 
>2. git add .
>3. git commit -m "注释语句"
>4. git push
